## 1. 全球主要厂商

### 1.1 IPG Photonics（美国）

| 维度 | 状态 |
|------|------|
| **成立时间** | 1990 年 |
| **总部** | 美国马萨诸塞州牛津市 |
| **全球地位** | 全球最大光纤激光器厂商，长期市场份额第一 |
| **脉冲产品线** | YLPN 系列（纳秒）、YLPF 系列（皮秒/飞秒） |
| **技术优势** | 全垂直整合（从光纤到激光器），专利布局深厚，功率与可靠性领先 |
| **市场份额** | 工业光纤激光器全球市占率约 30–40%（近年受国产冲击下降） |
| **2024 营收** | 约 9–10 亿美元 |
| **挑战** | 中国国产替代压力，价格竞争加剧 |

### 1.2 nLight（美国）

| 维度 | 状态 |
|------|------|
| **成立时间** | 2000 年 |
| **总部** | 美国华盛顿州温哥华市 |
| **定位** | 高功率半导体激光器 + 光纤激光器 |
| **脉冲能力** | 纳秒/皮秒脉冲光纤激光器 |
| **技术优势** | 半导体泵浦源自研（垂直整合泵浦芯片），亮度高 |
| **特色** | Alta™ 半导体激光器亮度行业领先 |
| **市场** | 北美、欧洲工业市场 |

### 1.3 SPI Lasers / II-VI / Coherent（英国/美国）

| 维度 | 状态 |
|------|------|
| **历史** | SPI Lasers 被 II-VI 收购，II-VI 更名为 Coherent |
| **脉冲产品** | RedPower 系列脉冲光纤激光器 |
| **特点** | MOPA 架构为主，灵活参数调节 |
| **市场** | 欧洲、亚洲工业加工市场 |

### 1.4 MKS Instruments / Spectra-Physics（美国）

| 维度 | 状态 |
|------|------|
| **产品** | Alpine 系列皮秒激光器，Mai Tai 系列飞秒振荡器 |
| **优势** | 超快激光器领域传统强者，科研市场占有率高 |
| **应用** | 微加工、科研光谱 |

### 1.5 NKT Photonics（丹麦）

| 维度 | 状态 |
|------|------|
| **定位** | 超连续谱 + 超快光纤激光器专家 |
| **产品** | aeroPULSE 飞秒激光器、SuperK 超连续谱 |
| **优势** | 飞秒光纤激光器商业化领先，光子晶体光纤技术 |
| **市场** | 科研 + 高端工业 |

## 5. 市场格局分析

### 5.1 全球工业光纤激光器市场
- **市场规模**：约 30–40 亿美元（2024 年），年增长率约 10–15%
- **趋势**：国产替代加速，中国厂商全球份额持续提升
- **价格**：持续下降，尤其是中低功率段

### 5.2 竞争态势
- **高端**（超快、超高功率）：IPG、Coherent、NKT 仍领先
- **中端**（纳秒脉冲、MOPA）：杰普特、锐科与国际品牌正面竞争
- **低端**（调 Q 打标）：已基本国产化，价格战激烈

### 5.3 产业链趋势
- **上游**：特种光纤（美国 Nufern/OFS）、泵浦半导体芯片仍是瓶颈
- **国产化率**：连续激光器 >80%，脉冲激光器约 60–70%，超快激光器 <50%
- **整合趋势**：下游设备厂商向上游激光器延伸
